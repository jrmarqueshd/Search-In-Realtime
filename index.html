Get Starter

Inclua antes de fechar a tag </body> :

<script src="https://cdn.jsdelivr.net/gh/jrmarqueshd/simple-and-fast-realtime-search-field/public/main.js"></script>

Defina em seu elemento <input /> :

<input search-field="on" type="search" />

Defina em seu elemento <ul> :

<ul list-search-id="on">
And ready, the magic will happen! NOTE: Elements within <ul> tag must be of type <li> for the best working
